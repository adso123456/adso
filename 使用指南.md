# Minecraft Agent 使用指南

## 准备工作

### 1. 安装 Minecraft Java 版

本工具需要 **Minecraft 1.20.1 / 1.21.1 Java 版**，推荐使用 HMCL 启动器：

- [HMCL 下载](https://hmcl.huangyuhui.net/download/)
- 安装后选择版本 1.20.1 / 1.21.1 即可

### 2. 获取 AI API Key

前往 [阿里云百炼平台](https://dashscope.aliyun.com/) 注册并获取免费 API Key。

## 启动

1. 复制 `.env.example` 为 `.env`
2. 编辑 `.env`，将 `DASHSCOPE_API_KEY` 改为你的 API Key
3. 双击 `启动.bat`，桌面控制窗口会自动弹出

启动后程序会自动开启所有服务，无需额外操作。

## 连接 Minecraft

1. 启动 Minecraft 1.20.1 / 1.21.1，进入一个世界
2. 打开局域网模式（按 ESC → 对局域网开放）
3. 机器人会自动连接并加入游戏

## 使用

进入游戏后，在聊天框发消息控制机器人：

| 怎么发 | 效果 |
|--------|------|
| `你好` | 闲聊对话 |
| `1砍3棵树给我` | AI 自动找树、砍伐、带回 |
| `1合成一把钻石镐` | 自动收集材料并合成 |
| `1去坐标 100 64 -200` | 移动到指定位置 |
| `1附近有没有钻石` | 搜索周围方块并汇报 |

> 以 `1` 开头触发 AI 指令执行，不加 `1` 则进入闲聊模式。

## 常见问题

**机器人不动？**
- 确保玩家在附近，机器人只能在玩家加载的区块内活动
- 等待约 20 秒，机器人会自动触发寻路脱困

**启动报错？**
- 检查 `.env` 文件是否存在且 API Key 正确
- 确保 3001、8000、7860 端口未被占用

**合成失败？**
- 确保所需原材料在背包中
- 复杂物品（如钻石装备）需要完整材料链
